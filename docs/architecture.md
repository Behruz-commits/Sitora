# Архитектура Sitora

**Sitora** состоит из трёх основных модулей:

1. **API (ядро)** — принимает запросы и формирует ответы.
2. **Bot (интерфейс)** — общается с пользователем.
3. **Data (данные)** — хранит кейсы, словари, ответы.

Будущие версии будут включать:
- Модуль обучения на локальных данных компаний Душанбе.
- Поддержку таджикского языка.
- Веб-интерфейс для демонстраций.
