# Sitora — Цифровой сотрудник Душанбе
Полный комплект для презентации: запуск API, Telegram-бота и Web-чата для демонстрации кейсов.

## Быстрый старт
1. pip install flask telebot requests
2. python api/main.py
3. python bot/telegram_bot.py
4. (по желанию) python bot/web_chat.py — открой localhost:9000

Примеры диалогов см. в /data/cases.md.
